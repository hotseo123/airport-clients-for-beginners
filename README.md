## 🧠 使用建议（新手必看）

- 只选 **一个客户端长期使用**，不要频繁更换  
- 出问题先 **换节点**，再考虑客户端  
- iOS 用户优先 **Shadowrocket**  
- Android 用户优先 **Clash for Android / Clash Meta**  
- macOS 用户优先 **ClashX / Clash Verge**  
- 建议提前准备 **一个备用客户端**

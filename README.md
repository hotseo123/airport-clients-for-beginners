## 🧠 新手使用建议

- **只选一个客户端长期用**，别频繁更换  
- 遇到问题先换节点，不要先怀疑客户端  
- iOS 用户优先 **Shadowrocket**  
- Android 用户优先 **FlClash / Clash Meta**  
- macOS 用户优先 **Clash Verge**  
- 建议提前准备 **1 个备用客户端**
